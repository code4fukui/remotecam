# remotecam
The English README is available here: [README.md](README.md)

Ayame WebRTCシグナリングサーバーを使用したリモートカメラビューアー。

## デモ
[https://code4fukui.github.io/remotecam/](https://code4fukui.github.io/remotecam/)

# 機能
- ブラウザでリモートカメラストリームを表示できます
- QRコードを使ってランダムなルームに自動的に接続できます
- 映像と音声の送受信をサポートしています

## 要件
なし、ブラウザで動作します。

## 使い方
1. [デモページ](https://code4fukui.github.io/remotecam/)を開きます。
2. QRコードをスキャンして、ランダムなルームに接続します。
3. リモートカメラのストリームが表示されます。
